Defra's Digital Waste Tracking API: terms

If you want [Waste receiver name 3] to use Defra's Digital Waste Tracking (DWT) API, you must accept the following terms.

! Warning You may lose your access and face fines or legal action if you misuse the API.

Your responsibilities

You must:

  • only collect and process data for the operation of your waste service and compliance with waste tracking regulations
  • notify the Information Commissioner's Office (ICO) and affected individuals if there is a breach involving personal data obtained via the API
  • ensure your software remains compatible with the latest version of the API

You must not:

  • misuse any data obtained through the API
  • share your API code with anyone you do not trust

Defra's role and responsibilities

  • may update the API and these terms at any time
  • will keep a record of the API code that identifies your waste receiver within the DWT service
  • reserves the right to revoke access to the API at any time for breach of these terms or for operational reasons
